# Language Change

[ApplicationLanguages.PrimaryLanguageOverride](https://docs.microsoft.com/en-us/uwp/api/windows.globalization.applicationlanguages.primarylanguageoverride)를 이용하여 앱을 재시작 하지 않고 언어를 변경하는 샘플 앱.

`ApplicationLanguages.PrimaryLanguageOverride`를 오버라이드 한 후 [ResourceContext.QualifierValues](https://docs.microsoft.com/en-us/uwp/api/windows.applicationmodel.resources.core.resourcecontext.qualifiervalues)의 `MapChanged` 이벤트를 리스닝 함.
